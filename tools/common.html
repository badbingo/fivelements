八字常见格局

了解八字格局，分析性格特征、人生追求与天赋潜能

八字格局概述

每个八字都会有1-3种格局，就像每个人都会有姓名、性别一样。通过判断八字的格局，可以分析出一个人的性格、人生追求、内心想法、特异功能和天赋等特征。

正八格

八字中最常见的八种基本格局：

  • 正财格、偏财格
  • 正印格、偏印格
  • 正官格、七杀格
  • 食神格、伤官格
  • 比肩劫财不入格局

外格

特殊格局，种类繁多：

  • 外格有几百种
  • 九龙先生会挑选常用、常见、断事准确的外格分享
  • 对于分辨一个八字会有很大帮助

分辨方法

判断格局的基本方法：

  1. 月支本气是否透干
  2. 月支中气是否透干
  3. 月支余气是否透干
  4. 三个都没有透干，取除日元外三个天干能量最大的为格局

正八格详解

以下是八种基本格局的特征分析，帮助您了解不同格局的性格特点和人生追求

偏财格

  • 整天想着做生意，干大事
  • 吃饭抢着付账，豪爽大方
  • 赌性大，投资喜欢一把梭哈
  • 锄强扶弱，大侠性格
  • 喜欢交朋友，请客吃饭，艳遇多
  • 收入不稳定，时有时无
  • 一心多用，同时做几个事情

正财格

  • 不愿意冒险，追求稳定收入
  • 一般为打工者居多
  • 钱来得不容易，勤俭节约
  • 诚实可靠，脚踏实地
  • 重视家庭，爱妻子
  • 正财太多，男人容易多次结婚
  • 追求：财富，男人喜欢追求女人

食神格

  • 会玩，心胸宽广不会发愁
  • 不喜与人争
  • 在那个宫位，那个身体部位就大
  • 身强食神泄秀，口才好，出口成章
  • 食神过旺，会有官司
  • 食神过多懒散，有拖延症
  • 头脑简单

伤官格

  • 官司口舌纷争比较多
  • 上进心强，与天斗，与地斗，与人斗
  • 喜欢跟领导对着干，不服管教
  • 个性独立，不喜欢被约束
  • 爱装逼，聪明伶俐
  • 杠精附体，喜欢抬杠
  • 恩怨分明，有仇必报
  • 不会道歉、死不认错

正印格

  • 热爱学习，心地善良
  • 重情重义
  • 有艺术细胞，喜欢琴棋书画
  • 代表权利
  • 印为护身之物，一生没有大灾大难
  • 一生多贵人相助

偏印格

  • 心细如发，有专业技能
  • 一技傍身，但忧愁抑郁
  • 多疑，不信任别人
  • 能看到别人看不到的细节
  • 第六感强、敏感
  • 偏印多能看到鬼
  • 适合作军师、参谋，脑子不停思考

正官格

  • 一身正气，光明磊落
  • 容易当领导，老大领袖
  • 对自己严格要求
  • 交朋友做减法，与有恶习之人不交往
  • 厚道，认真，古板，较真
  • 过于严肃，强迫症
  • 爱操心，劳累

七杀格

  • 病神，心狠手辣，胆子大
  • 聪明伶俐
  • 执行力强，嫉恶如仇
  • 性格火爆，翻脸如翻书
  • 女命之情人
  • 鲁莽暴躁，执行力强
  • 追求：尊重，名声地位

比肩格

  • 身弱跟人合作能发财
  • 一生能得到兄弟姐妹帮助
  • 身强，给人合作会破财
  • 一生不能得到兄弟姐妹帮助
  • 身弱比肩可帮身，身旺比肩夺财
  • 不会道歉，死不认错
  • 要面子，朋友多

劫财格

  • 赌博，好喝酒，好色
  • 见面熟，情商高
  • 见人说人话，见鬼说鬼话
  • 会吹牛
  • 喜欢借别人的钱
  • 男命劫财多克妻
  • 追求：平平安安，顺其自然
