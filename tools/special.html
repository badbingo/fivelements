八字特殊格局

探索八字命理中的特殊格局，了解富贵贫贱的命理奥秘

三奇贵人格

三奇贵人格是命格中最好的格局之一，拥有此格局者多为非凡之人

三奇贵人解析

三奇贵人是指"天奇"、"地奇"和"人奇"三种特殊组合。这种命格在八字中很少见，物以稀为贵。三奇命人绝对不是一般人，命带三奇者，总是博学多才，一生抱负极大，胸襟宽阔，有功名，有成就，是大富大贵之人。

三奇类型 天干组合 特点
天上三奇 乙丙丁 天干连续出现乙丙丁，主贵
地上三奇 甲戊庚 天干连续出现甲戊庚，格局最高
人间三奇 辛壬癸 天干连续出现辛壬癸，主富

古书记载：三奇拱向紫微宫，最喜人生命里逢。命理阴阳真宰相，功名富贵不雷同。

三奇贵人格注意事项

  1. 三奇隔断，贵气受损 - 三奇天干被其他天干隔开，格局层次降低
  2. 顺序可以颠倒，打乱的话贵气受损 - 如乙丁丙也算三奇，但不如乙丙丁纯粹
  3. 地上三奇甲戊庚格局最高，必是国家栋梁 - 此组合最为尊贵
  4. 三奇亦要地支有合而贵，三会，三合，六合 - 地支配合很重要
  5. 天干三奇成，地支遇刑冲破害，贵气受损 - 地支有破坏则减分

禄元互换格

中年成大事儿，大富大贵的格局

古书记载：禄元互换有四天，生时必要透正官；不逢冲破与伤害，贫穷离乡富贵还。

丙子日 癸巳时 丙禄在巳，子与巳绝，癸巳为官
庚子日 丁亥时 庚禄在申，子与申合，丁亥为官

六十甲子在循环，禄元互换有四天。一十五天出一个，位及丞相富贵格。中年以后可以生出无数的正官正财，所以也是非常好的格局。

天元一气格

天元一气格也出过很多皇帝，大人物，命运较为奇特

年柱、月柱、日柱、时柱这四柱都是一样的，命运较为奇特。天元一气格的八字，总共有10组，分别是：壬寅壬寅壬寅壬寅、辛卯辛卯辛卯辛卯、庚辰庚辰庚辰庚辰、己巳己巳己巳己巳、戊午戊午戊午戊午、丁未丁未丁未丁未、丙申丙申丙申丙申、乙酉乙酉乙酉乙酉、甲戌甲戌甲戌甲戌、癸亥癸亥癸亥癸亥。

此命为奇格，很少见到，但八字五行缺少2-3样。古书讲：人之八字，最宜四柱流通，五行生化；大忌四柱缺陷，五行偏枯。虽然富足但孤傲而且寿短。

历史名人案例

  • 成吉思汗的八字：乙亥、乙酉、乙酉、乙酉 - 征战四方，一代暴君
  • 隋炀帝的八字：乙酉、乙酉、乙酉、乙酉 - 在位施暴政，最后五十岁时死于兵变
  • 杨贵妃的八字：乙酉、乙酉、乙酉、乙酉 - 早年富贵，结局凄惨
  • 唐太宗的八字：庚辰、庚辰、庚辰、庚辰 - 为一代明主，开启贞观之治，但死时年仅五十一岁

另外还有天干一气格：四个天干一样，地支不一样；地支一气格：四个地支一样，天干不一样。贵气不如天元一气格。

杀印相生格

八字最高格局，王者格局

古书讲：杀生印，印生身，富贵双全，杀不离印。印不离杀，杀印相生，功名显达。看八字主要看七杀，七杀本为偏官，但因性质凶狠，索人性命古人用杀来命名。大运流年遇多个七杀无制，断为一个人的死期。

克制七杀的方法有两个：身弱印枭来化杀，身强食伤来克制杀。七杀代表执行力强、霸道、脾气火爆、耿直、刚正。正印代表权力、心地善良、喜欢学习。杀印相生格文武兼备，功勋粲然，出将入相。

杀印相生格成立条件

  1. 日元必须弱
  2. 本命局有七杀，有正印
  3. 本命局不要有刑冲破害
  4. 印与杀的比例要成正比（1个正印可以化1个七杀）
  5. 男命乾造需要阴日干，女命坤造需要阳日干
  6. 遇大运流年有强财坏印，七杀无制易出灾祸
  7. 日元地支需要有根，微根都行，无根易短寿
  8. 多柱都有正印最好

人见人爱的人民币上印的那位伟大的爷爷就是杀印相生格。

食神制杀格（七杀食制格）

此为贵格，十有九贵，为中上等格局

这个格局也出过黄帝，但多为中高层管理者、武将、官职，所以称为贵格。韩信、黄兴、美国总统奥巴马都是食神制杀格。

七杀为四凶神之首，代表病痛，执法单位找事儿，小人等不好的信息，见到了一定要制服。食神制杀格局，是指一个人本身身强，此时印为忌凶，命中七杀太旺。可以派出喜用食神、伤官进行压制七杀，不过伤官有狂傲之气，伤官制杀，容易惹出是非，不如食神制杀效果好。

食神是一个吉神，食神为财根，可以带来财富，在命局中可以制住七杀，使其不能伤害到命主本身，所以叫做食神制杀格。食神制服了七杀，使这个人的命格变好，从一个暴力的人变成了一个认真讲诚信而且很有能力的人，可以说是一种非常不错的格局。

食神制杀格成立条件

  1. 必须身强，食神制杀格的前提就是日主要身强
  2. 身强，杀旺，食旺，才能成格
  3. 注意七杀与食神的比例，平衡最好
  4. 逢大运克泄耗过多，日元从身强变身弱，食神七杀变成忌凶
  5. 日元邀请食神制七杀，本命局不能见到偏印
  6. 大运流年见到偏印克制食神，破格，七杀无制易出灾祸
  7. 食神不能被合化，被冲，否则七杀无制必应灾
  8. 本命局不要有刑冲破害，否则贵气减少

身杀两停格

在所有成格之七杀格中，尤为上上等富贵者

入格者，定为人中龙凤、富贵双全、风云翘楚人物!大富大贵之命。为什么身杀两停格会富贵？一个人最好的发财状态是均衡偏强一点。身弱发不了财。日元太强容易狂妄自大，不听劝告，刚愎自用，胡乱投资，盲目扩张，反而容易损财。

一个八字日元和七杀的能量差不多，日元时刻处于一种均衡的状态，大运流年遇到财星，日元去克财，就比较容易发财。另外七杀代表心狠手辣，做事有执行力。没有七杀的人心太善，发不了大财。

身杀两停格特点

  1. 日元与七杀的能量差不多
  2. 日元遇到大运流年克泄耗过多，变为身弱，七杀无制，易出灾祸
  3. 本命局七杀能量大，日元身弱大运流年喜见正印化杀，变为杀印相生格
  4. 本命局日元能量大，七杀弱，大运流年喜见食神，变为食神制杀格
  5. 本命局不要有刑冲破害，否则贵气减少

伤官配印格

没有一个成大事儿的八字是全部吉神的，所有伟人八字中一定有伤官或七杀

如果八字组合得当，恶神被制住，命主才能成就大事儿。伤官配印格容易成名，成龙、徐静蕾、陈好、周总理都是此格局。

伤官代表官司口舌纷争，伤官重则为自己有血光之灾。性格方面：坏的一面是狂妄自大，认为老子天下第一，谁也不服。不负管教，抗上一身傲骨，自视不凡，持才傲物，心高气傲，锋芒毕露。好的一面是积极开拓，上进心强，争强好胜，聪明伶俐，企图心强，越挫越勇，有困难第一个上，多才多艺，口才好。

伤官旺的人不服管教，但听人劝，可以引导，可以感化。世间一物降一物，印的特点就是慈爱、奉献、忍耐、重感情、爱心、有修养、慈祥。伤官旺的服软不服硬，可以以德感化，所以要用印来制伤。你想想孙悟空本事那么大，为什么愿意辅佐唐僧这个没本事的领导？就明白了伤官配印格的原理。

伤官配印格条件

  1. 日元身弱，伤官为忌凶，印为喜用，以印制伤，才能成格
  2. 伤官配印格，忌见财星，大运流年财星坏印，伤官无制，易出口舌官司
  3. 伤与印的距离要得当，离得太远，印制不住伤
  4. 偏印也能制服伤官，但是没有正印能量纯粹
  5. 日元有根，不能太弱，否则短寿
  6. 本命局有刑冲破害，格局贵气受损

伤官生财格

聪明不过伤官，伶俐不过七杀

伤官代表聪明，代表上进心，不服输的精神。八字日元身强，伤官旺，财旺，必是富贵之人。伤官配印主贵。伤官生财主富。

伤官生财格条件

  1. 日元身强。伤官为喜用，财才为喜用，以伤生财才，泄掉伤官恶神之气，方能成格
  2. 日元身强。伤官旺，八字无财才，虽然聪明伶俐，但必然贫穷。大运流年遇财也有发财的机会
  3. 日元身强。财才旺，八字无食伤，满脑子想着赚钱，没有赚钱的手段，大运流年遇到食伤也有发财的机会
  4. 女命伤官生财格，也算贵格，是女强人。董明珠开大会口出狂言
  5. 伤官生财格最忌大运流年见官，伤官见官为祸百端

伤官见官格

此格局为凶格，只做介绍，不建议采用

伤官见官，祸事百端。伤官代表好胜逞强，不服管制，自命不凡，对谁都看不上眼，高傲轻高。官是代表上级部门，国家法律法规，是对个人的一种限制行为，所以伤官旺，一见到官。上级和法律就会管你制你，你跟管理你的人对着干，自然就有灾祸。

会出现丢职，受领导批评，工作受制，下岗，或者官灾诉讼，女同志与丈夫不和等事情。

伤官见官格特点

  1. 日元身强，官为喜用，此时伤官见官，以凶论，容易出灾祸
  2. 日元身弱，官为忌凶，此时伤官见官，不以凶论
  3. 伤官见杀不以伤官见官论
  4. 日元身强，本命局伤官旺，不见正官，为伤官伤尽。四柱中伤官伤尽，岁运逢官星吉利

伤官泄秀格

日元身强情况下，全盘只有比劫，印枭和伤官

伤官得用，可以把自身的能力发挥出来，也是贵格之一。必为聪明绝顶之人，才华横溢，多为艺术表演和文人学术之人。

伤官泄秀格与伤官配印格的成格条件基本相同。唯一不同的是一个身强一个身弱。身弱伤官配印格主贵；身强伤官泄秀格主艺术。
